The data pipeline converts raw multilingual sources into cleaned, normalized, and split datasets, then builds unified vocabulary packs. Outputs are designed to feed directly into training via ModernParallelDataset.

How to Run

1. Downloader
python -m data.unified_data_downloader --config config/base.yaml
2. Pipeline (dry-run)
python -m data.unified_data_pipeline --dry-run --config config/base.yaml
3. Full Pipeline
python -m data.unified_data_pipeline --config config/base.yaml

Use --dry-run to synthesize small local datasets for a quick end-to-end validation.

Files referenced above are clickable when hosted; in local dev, navigate to the exact paths.
