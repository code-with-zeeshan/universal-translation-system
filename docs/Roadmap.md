## Roadmap

### Phase 1: Core System (Current)
- Universal encoder with dynamic vocabularies
- Cloud decoder with GPU optimization (FastAPI/uvicorn)
- SDKs for Android, iOS, Flutter, React Native, Web (under `sdk/`)
- Support for 20 languages
- Docker Compose + Kubernetes + Helm deployment
- Redis integration with disk mirroring
- 285+ tests across 14 test files
- Production scripts (install, build, setup)

### Phase 2: Production Hardening
- Full offline mode with compressed models
- Voice translation support
- Real-time conversation mode
- Support for 50+ languages
- Enhanced monitoring dashboards

### Phase 3: Community Cloud
- Self-service decoder deployment
- Community-contributed decoder pool
- Decoder resource sharing marketplace
- Usage analytics dashboard

### Future Ideas
- Offline mode with compressed models
- Voice translation support
- Real-time conversation mode
- Support for more languages (goal: 50+)
